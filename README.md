## Where do we get the data?
1. Elected officials
	1. precinct
	2. zip code
	3. office responsibilities
	4. campaign messaging
2. Issues
	1. taxes, abortion, zoning reform, etc.
	2. wording heterogeneity
		1. text corpus of similar references to common issues
		2. aliases for each
